Media Storage Protocol

Important: To ensure proper media usage, please do not share any files with anyone outside of the TDAV Media team. This is crucial for protecting against copyright issues and ensuring a respectful portrayal of our students and university.

Media Storage Location(s)

Media will be stored in the following locations:

  1. Master SSD in the Media Supervisor's Office
  2. Cloned SSD in the Box Office
  3. InterPlanetaryFileSystem or similar for Backup Archive.Read More

Please do not remove these drives, but rather keep them secured in their locations preferably k-locked, and pull files from them with your work computer instead.

Media will no longer be stored on cloud Drives (ie. google drive) in order to regulate usage and keep costs down.

For now, smaller items such as rehearsal/seminar recordings should be stored locally on faculty/staff computers, and can be shared either using a thumb drive. By the next semester, I plan to migrate our media sharing entirely to a secure IPFS network (or similar).

Also as a note, do notupload any media to NMU's Canto repository unless granted explicit permission.

File Organization Instructions

All files are stored in the master folder titled TDAV MEDIA DRIVE. Within this folder, create subfolders for each production, labeled by the show title.

File Naming Convention

When saving files, adhere to the following format:

  • Date: Use YYMMDD format.
  • Show Title: Include the title of the show.
  • File Type: Indicate the type of file with the following suffixes:
    • _RAW for raw files (photos or videos)
    • _EDIT for edited projects
    • _DELIVER for deliverables

TRASHME Folder

Use the TRASHME folder for items scheduled for deletion:

  • Label this folder as TRASHME YYMMDD.
  • Items will be kept until the specified date for regular cleanup.

Example File Names

Here are some examples of properly formatted file names:

  • 241101 Jekyll and Hyde - Photo_RAW
  • 241101 Les Miserables - Video_DELIVER

Conclusion

Following these instructions carefully will ensure proper organization and easy access for our media team. Thank you for your cooperation!
