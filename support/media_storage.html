File Organization Instructions

Folder Structure

All files are stored in the master folder titled TDAV MEDIA DRIVE.

Within this folder, create subfolders for each production, labeled by show title.

File Naming Convention

When saving files, use the following format:

  • DATE LABELING: Use YYMMDD format for the date.
  • SHOW TITLE: Include the title of the show.
  • FILE TYPE: Indicate the type of file with the following suffixes:
    • _RAW for raw files (photos or videos)
    • _EDIT for edited projects
    • _DELIVER for deliverables

TRASHME Folder

Use the TRASHME folder for items that are scheduled to be deleted.

  • Label this folder as TRASHME YYMMDD.
  • Items will be kept until the specified date for regular cleanup.

Example File Names

Here are some examples of properly formatted file names:

  • 241101 Jekyll & Hyde - Photo_RAW
  • 241101 Les Miserables - Video_DELIVER

Conclusion

Please follow these instructions carefully to ensure proper organization and easy access for our media team. Thank you!
