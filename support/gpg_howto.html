Encrypting Files with GPG

Step 1: Install GPG

For Windows: Download Gpg4win and follow the installation steps.

For macOS: Use Homebrew. Open Terminal and run: brew install gnupg or download from GPG Suite.

For Linux: Open a terminal and run: sudo apt-get install gnupg

Step 2: Generate Your GPG Key Pair

Open your terminal or command prompt and run:

gpg --full-generate-key
  • Select the key type (default is usually fine).
  • Choose a key size (2048 or 4096 bits recommended).
  • Set an expiration date (optional).
  • Enter your user ID information (name and email).
  • Set a passphrase to protect your key.

Step 3: Encrypt a File

To encrypt a file, use the following command:

gpg -e -r recipient@example.com filename
  • -e: Encrypt the file.
  • -r: Specify the recipient's email associated with their GPG key.
  • filename: The file you want to encrypt.

Step 4: Share the Encrypted File

Now, you can safely share the encrypted file on the P2P network.

Step 5: Decrypting a File

When you receive an encrypted file, you can decrypt it using:

gpg -d filename.gpg > decrypted_file

Conclusion

Using GPG ensures that your files are encrypted and secure before sharing them on a P2P network. Make sure to keep your private key and passphrase safe!
