Rocky Horror 2024 - Media Design

For the 2024 production of The Rocky Horror Picture Show, I am using HTML with embedded CSS styling to craft atmospheric animations that enhance the visual storytelling. HTML is the foundation for most websites, while CSS is the language used to design the front-end interface. As an interesting challenge, I am creating these animations entirely with HTML and CSS to demonstrate that, with just a few lines of code and a basic text editor, you can create visually interesting and useful animations. The animations are designed to complement the set design and overall mood, blending seamlessly with the scenery rather than overpowering it. By integrating subtle, dynamic visuals, the goal is to create an immersive environment that supports the narrative without overshadowing the live performance.
