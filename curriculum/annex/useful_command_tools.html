Homebrew

Install Homebrew

    /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

Homebrew is a package manager for macOS that simplifies the installation of software.

GitHub

Install Git

    brew install git

Installs Git, a version control system to manage source code.

Clone a Repository

    git clone <repository-url>

Creates a local copy of a Git repository from the specified URL.

Check Status

    git status

Displays the current status of the repository, including staged, unstaged, and untracked files.

Commit Changes

    git add .
    git commit -m "Your commit message here"

Adds changes to the staging area and commits them to the repository with a message describing the changes.

FFMPEG

Install ffmpeg

    brew install ffmpeg

Installs FFmpeg, a powerful tool for handling multimedia data.

Compress A Video

    ffmpeg -i input.mp4 -vcodec libx264 -crf 28 output.mp4

Compresses a video file while maintaining reasonable quality. The lower the CRF value, the higher the quality.

Segment A Video

    ffmpeg -i /Volumes/Media/HyperDeck_0002.mov -f segment -segment_time 60 -c copy /Volumes/Media/output%03d.mp4

Splits a video into segments of a specified duration (in seconds).

Shard A File (Video)

    ffmpeg -i input.mp4 -c copy -map 0 -segment_time 60 -f segment output%03d.mp4

Similar to segmenting, this command allows you to break down a video file into smaller chunks.

Convert Video Format

    ffmpeg -i input.mov output.mp4

Converts a video from one format to another.

Extract Audio from Video

    ffmpeg -i input.mp4 -q:a 0 -map a output.mp3

Extracts audio from a video file and saves it as an MP3 file.

IPFS

Install IPFS

    brew install ipfs

Installs the IPFS command-line tool on your system, enabling you to interact with the IPFS network.

Initialize IPFS

    ipfs init

Sets up a new IPFS node, creating necessary configuration files and a unique node identity.

Start the IPFS Daemon

    ipfs daemon

Runs the IPFS node, allowing you to interact with the IPFS network and share files.

Add a File to IPFS

    ipfs add <file-path>

Uploads a specified file to the IPFS network and returns its content hash, which can be used to retrieve the file later.

Retrieve a File from IPFS

    ipfs cat <file-hash>

Fetches and displays the contents of a file from IPFS using its unique content hash.

List All Files in IPFS

    ipfs ls

Displays the files currently stored in your local IPFS node, including their hashes.

Check IPFS Node Status

    ipfs id

Shows the identity and configuration details of your IPFS node, including your node's peer ID.

Pin a File to Keep it on Your Node

    ipfs pin add <file-hash>

Ensures that a specified file remains available on your node, preventing it from being garbage collected and deleted.

Unpin a File

    ipfs pin rm <file-hash>

Removes the pin from a file, allowing it to be garbage collected from your node.

Check the Status of Pinned Files

    ipfs pin ls

Lists all files that are currently pinned to your IPFS node.

Publish a File to IPNS

    ipfs name publish <file-hash>

Publishes a file to the InterPlanetary Naming System (IPNS), allowing it to be accessed via a human-readable name.

Resolve an IPNS Name

    ipfs name resolve <ipns-name>

Fetches the current content hash associated with a given IPNS name, allowing you to retrieve the file.

Get Peer Connections

    ipfs swarm peers

Displays a list of peers your IPFS node is currently connected to, showing the health and reachability of the network.
