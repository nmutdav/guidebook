Syllabus TH-242: Introduction to Sound and Media Design

  • Offered: Fall, even years
  • Room: McClintock Room 125-O
  • Time: Mondays and Wednesdays from 8:00am to 9:40am
  • Instructor: Dominic Mrakovcich

Course Description

TH-242 provides a comprehensive introduction to sound and media design using Processing IDE and SuperCollider. The course emphasizes foundational skills in sound and media design, focusing on creating design concepts from simplified parameters. Students will develop, present, and execute design proposals applicable to various media contexts, including theatre, film, and multimedia projects.

Notes: This is an eight-week course.

What are Processing IDE and SuperCollider?

Processing IDE is a flexible software platform used to develop visual art, animations, and interactive projects. It provides an intuitive interface for coding and includes numerous built-in libraries for graphics, sound, and interaction.

SuperCollider is a platform for audio synthesis and algorithmic composition. It is used for creating complex audio applications and real-time sound processing. Students will learn to use SuperCollider for sound synthesis, sampling, and interactive audio applications.

Application in Sound and Media Design:

  • Processing IDE: Useful for interactive media and rapid prototyping of visual elements.
  • SuperCollider: Ideal for advanced sound synthesis and real-time audio manipulation.

Tracks & Objectives

1. Sound Design with SuperCollider

Develop proficiency in using SuperCollider for sound synthesis and interactive audio applications.

  • Sound Fundamentals: Understand sound principles and audio processing within SuperCollider.
  • Interactive Sound Design: Create interactive sound environments and applications.
  • Audio Synthesis: Implement techniques for advanced sound synthesis and manipulation.
  • Project Execution: Complete a sound design project using SuperCollider.

2. Media Design with Processing IDE

Acquire skills in designing media such as animations, interactive visuals, and multimedia projects using Processing IDE.

  • Media Design Principles: Master core concepts for graphics and animations in Processing.
  • Interactive Media: Develop media elements responsive to user input.
  • Visual Storytelling: Create cohesive and compelling media projects.
  • Project Execution: Complete a media design project using Processing IDE.

3. Design Proposals and Deliverables

Focus on the creation, presentation, and execution of design proposals, ensuring effective communication and practical application.

  • Proposal Crafting: Develop and refine design proposals for sound and media projects.
  • Execution: Produce deliverables and execute design proposals effectively.
  • Presentation Skills: Communicate design concepts and their execution effectively.

Grading Breakdown

  • Final Project & Proposal Execution: 30%

Grading Scale:

  • A (90% - 100%): Exceptional performance with high-quality work and innovative solutions.
  • B (80% - 89%): Above-average performance with solid understanding and proficient work.
  • C (70% - 79%): Satisfactory performance meeting average standards.
  • D (60% - 69%): Below-average performance with notable deficiencies.
  • F (0% - 59%): Unsatisfactory performance failing to meet minimum requirements.
