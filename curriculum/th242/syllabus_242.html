Syllabus TH-242: Introduction to Sound & Media Design

  • Offered: Fall, even years
  • Room: McClintock Room 125-O
  • Time: Mondays and Wednesdays from 8:00am to 9:40am
  • Instructor: Dominic Mrakovcich

Course Description

TH-242 is a hands-on course focusing on sound and media design using the Processing IDE. The course introduces fundamental concepts and techniques for creating interactive and visually compelling projects. Students will explore the capabilities of Processing to develop multimedia applications that integrate sound and visual elements.

Notes: This is an eight-week course.

What is Processing IDE?

Processing IDE is a versatile platform for developing visual art, animations, and interactive media. It provides an intuitive environment for coding and includes built-in libraries for graphics, sound, and interaction, making it ideal for creating complex visual and audio projects.

Application in Sound and Media Design:

  • Visual Design: Create and manipulate visual elements to produce animations and interactive graphics.
  • Audio Integration: Integrate sound with visual components to develop immersive multimedia experiences.

Course Tracks & Objectives

1. Sound Design with Processing

Develop skills in integrating and manipulating sound within visual projects using Processing.

  • Sound Fundamentals: Learn to manage and play audio files within Processing.
  • Interactive Sound Design: Create sound elements that respond to user interactions or visual changes.
  • Audio Visualization: Develop visual representations of audio data, such as spectrograms or waveform displays.
  • Project Execution: Complete a sound design project incorporating audio elements into visual projects.

2. Media Design with Processing

Acquire proficiency in creating multimedia content including animations and interactive visuals using Processing IDE.

  • Media Design Principles: Master core concepts of animation, graphics, and user interface design in Processing.
  • Interactive Media: Develop projects where visual elements respond to user input or sound.
  • Visual Storytelling: Craft engaging visual narratives and interactive experiences.
  • Project Execution: Complete a media design project that demonstrates the integration of visual and audio elements.

3. Design Proposals and Deliverables

Focus on creating, presenting, and executing design proposals for multimedia projects.

  • Proposal Crafting: Develop and refine design proposals that effectively integrate sound and visual elements.
  • Execution: Produce deliverables that meet the specifications of your design proposals.
  • Presentation Skills: Communicate your design concepts and project outcomes effectively.

Grading Breakdown

  • Attendance: 20%

Grading Scale:

  • A (90% - 100%): Exceptional performance with high-quality work and innovative solutions.
  • B (80% - 89%): Above-average performance with solid understanding and proficient work.
  • C (70% - 79%): Satisfactory performance meeting average standards.
  • D (60% - 69%): Below-average performance with notable deficiencies.
  • F (0% - 59%): Unsatisfactory performance failing to meet minimum requirements.

Notes:
