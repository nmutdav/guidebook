Syllabus TH-242: Introduction to Sound and Media Design

  • Offered: Fall, even years
  • Room: McClintock Room 125-O
  • Time: Mondays and Wednesdays from 8:00am to 9:40am
  • Instructor: Dominic Mrakovcich

    • Course Description

    TH-242 provides a comprehensive introduction to sound and media design using the Processing IDE. The course emphasizes foundational skills in both sound and media design, with a focus on creating design concepts from simplified parameters. Students will learn to develop, present, and execute design proposals applicable to various media contexts including theatre, film, and multimedia projects.

    Notes: This is an eight-week course.

    What is Processing IDE?

    Processing IDE is a flexible software platform used to develop visual art, animations, and interactive projects. It is widely used in creative coding due to its ease of use and accessibility. The environment provides an intuitive interface for writing code and includes numerous built-in libraries for graphics, sound, and interaction.

    Key Features:

    • Visual Feedback: Immediate graphical output of code, allowing for rapid iteration.
    • Simple Syntax: Designed to be easy to learn for artists and non-programmers.
    • Extensive Libraries: Includes libraries for graphics, sound, and user interaction.
    • Community Support: A large community and many resources for learning and troubleshooting.

    Application in Sound and Media Design:

    Processing IDE is particularly useful in an introductory course on sound and media design for several reasons:

    • Ease of Learning: Its simple syntax and visual feedback make it accessible for beginners, allowing students to quickly grasp the basics of coding for sound and media projects.
    • Interactive Projects: The IDE supports interactive media, enabling students to create projects that respond to user input, a crucial aspect of modern sound and media design.
    • Rapid Prototyping: The immediate graphical output facilitates rapid prototyping, helping students experiment with different design concepts and iterate on their ideas efficiently.

    Applicability to Theatre and Other Fields:

    Processing IDE is not only valuable in a classroom setting but also has broad applications in various fields:

    • Theatre: In theatre, Processing IDE can be used to design interactive stage visuals, soundscapes, and lighting effects that enhance the audience's experience.
    • Film and Multimedia: It supports the creation of dynamic visuals and soundtracks for film and multimedia projects, allowing for creative exploration and innovative design solutions.
    • Art Installations: Artists and designers use Processing IDE to develop interactive art installations that respond to environmental factors or audience interactions.
    • Educational Tools: Its user-friendly nature makes it an excellent tool for teaching coding concepts and design principles in various educational settings.

    Tracks & Objectives

    1. Sound Design with Processing IDE

    Develop proficiency in using Processing IDE for creating sound experiences, including visualizations and interactive audio applications.

    • Sound Fundamentals: Understand sound principles and audio processing within Processing.
    • Interactive Sound Design: Create interactive sound environments and applications.
    • Audio Visualization: Implement techniques for visualizing audio data.
    • Project Execution: Complete a sound design project using Processing IDE.

    2. Media Design with Processing IDE

    Acquire skills in designing media such as animations, interactive visuals, and multimedia projects using Processing IDE.

    • Media Design Principles: Master core concepts for graphics and animations in Processing.
    • Interactive Media: Develop media elements responsive to user input.
    • Visual Storytelling: Create cohesive and compelling media projects.
    • Project Execution: Complete a media design project using Processing IDE.

    3. Design Proposals and Deliverables

    Focus on the creation, presentation, and execution of design proposals, ensuring effective communication and practical application.

    • Proposal Crafting: Develop and refine design proposals for sound and media projects.
    • Execution: Produce deliverables and execute design proposals effectively.
    • Presentation Skills: Communicate design concepts and their execution effectively.

    Plagiarism Policy

    While studying together is encouraged, all submitted work must be original. Any use of uncredited sources, copying of another student’s work, or submission of work that is not your own will be considered plagiarism and will be subject to disciplinary action.

    In addition, the use of AI or similar tools (such as chatbots, code generators, or content generators) to complete assignments, projects, or any other coursework is strictly prohibited unless explicitly allowed by the instructor. All submitted work must reflect your own understanding, effort, and creativity. Submitting work generated by AI tools without proper acknowledgment will be treated as academic dishonesty.

    Refer to the university’s plagiarism policy and consult the Writing Center or instructor if you need clarification on what constitutes plagiarism or improper use of tools.
