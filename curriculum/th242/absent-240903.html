Instructions for Completing and Submitting Your Report

  • 3. Upload the Report to GitHub:
    • Log in to your group's GitHub repository.
    • Navigate to the appropriate folder (e.g., `reports/`).

    • Paste your Markdown report into this file.
    • Commit the file with a message such as "Added report for September 3, 2024 breakout session."

If you encounter any issues with GitHub or need assistance, reach out to your group members. Make sure the report is submitted by the end of the class period.
