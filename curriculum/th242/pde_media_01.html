Tutorial 1: Introduction to Processing

Objective: Get familiar with the Processing environment and basic concepts.

1. Overview of Processing

Processing is a flexible software sketchbook and a language for learning how to code within the context of the visual arts. It was created to simplify the process of coding for visual projects and make programming accessible for artists and designers.

2. Setting Up Processing

  • Download Processing from Processing.org.
  • Install it following the instructions for your operating system.

3. Your First Sketch

Open Processing and create a new sketch. A Processing sketch is a single file with a .pde extension that contains your code.


void setup() {
    size(800, 600);  // Define the size of the window
    background(200); // Set the background color
}

void draw() {
    ellipse(400, 300, 100, 100); // Draw an ellipse at the center of the window
}

Explanation:

  • void setup(): Initializes the sketch; this function runs once when the program starts.
  • size(800, 600): Sets the dimensions of the window, crucial for your visual space.
  • background(200): Sets a gray background color to start with.
  • void draw(): Continuously executes the code inside it, updating the display.
  • ellipse(400, 300, 100, 100): Draws a circle in the center of the window.

This tutorial introduces the basic structure of a Processing sketch and the fundamental functions for creating simple graphics.

Tutorial 2: Basic Drawing Functions

Objective: Learn basic drawing functions and how to create shapes.

1. Drawing Shapes


void setup() {
    size(800, 600);
    background(255);
}

void draw() {
    fill(255, 0, 0);    // Set the fill color to red
    rect(100, 100, 150, 100); // Draw a rectangle
    fill(0, 255, 0);    // Set the fill color to green
    ellipse(400, 300, 100, 100); // Draw a circle
    fill(0, 0, 255);    // Set the fill color to blue
    triangle(600, 150, 550, 300, 650, 300); // Draw a triangle
}

Explanation:

  • fill(255, 0, 0): Sets the fill color for shapes. You can use RGB values to choose colors.
  • rect(100, 100, 150, 100): Draws a rectangle at the specified position with the given width and height.
  • ellipse(400, 300, 100, 100): Draws an ellipse at the specified position with the given width and height.
  • triangle(600, 150, 550, 300, 650, 300): Draws a triangle with vertices at the specified coordinates.

Understanding these basic drawing functions is essential for creating visual elements and compositions in your sketches.

Tutorial 3: Working with Colors

Objective: Learn how to use and manipulate colors in Processing.

1. Using Colors


void setup() {
    size(800, 600);
}

void draw() {
    background(255); // Set the background color to white
    
    // Draw with different colors
    fill(255, 0, 0); // Red
    ellipse(200, 300, 100, 100);
    
    fill(0, 255, 0); // Green
    rect(350, 250, 100, 100);
    
    fill(0, 0, 255); // Blue
    triangle(500, 300, 450, 450, 550, 450);
}

Explanation:

  • fill(255, 0, 0): Sets the fill color using RGB values. Here, it's red.
  • fill(0, 255, 0): Sets the fill color to green.
  • fill(0, 0, 255): Sets the fill color to blue.
  • background(255): Sets the background color to white, ensuring the shapes are visible.

Color manipulation is key for creating visually appealing and distinct elements in your sketches.

Tutorial 4: Understanding Coordinates

Objective: Learn about the coordinate system in Processing and how to position shapes.

1. Coordinate System Basics


void setup() {
    size(800, 600);
    background(255);
}

void draw() {
    fill(255, 0, 0);
    ellipse(mouseX, mouseY, 50, 50); // Draw an ellipse at the mouse position
}

Explanation:

  • mouseX: Provides the current x-coordinate of the mouse pointer.
  • mouseY: Provides the current y-coordinate of the mouse pointer.
  • ellipse(mouseX, mouseY, 50, 50): Draws an ellipse at the mouse location, demonstrating how to use coordinate values.

Understanding coordinates is essential for positioning elements accurately and creating interactive sketches.

Tutorial 5: Basic Animation

Objective: Create simple animations by updating the position of shapes over time.

1. Creating Basic Animation


float x = 0;

void setup() {
    size(800, 600);
    background(255);
}

void draw() {
    background(255);
    fill(0, 100, 255);
    ellipse(x, height / 2, 50, 50); // Draw an ellipse moving horizontally
    x += 2; // Increment x to animate the ellipse
    if (x > width) {
        x = 0; // Reset x to 0 if it goes off-screen
    }
}

Explanation:

  • float x = 0: Initializes the x-coordinate for the ellipse.
  • x += 2: Updates the x-coordinate each frame, creating movement.
  • if (x > width): Resets the x-coordinate when the ellipse moves off-screen, creating a continuous loop.

Animating shapes is fundamental for creating dynamic and engaging visual experiences.

Tutorial 6: Using Functions

Objective: Learn how to create and use functions to organize and reuse code.

1. Defining and Calling Functions


void setup() {
    size(800, 600);
}

void draw() {
    background(255);
    drawEllipse(400, 300, 100); // Call the custom function to draw an ellipse
}

void drawEllipse(float x, float y, float diameter) {
    fill(0, 100, 255);
    ellipse(x, y, diameter, diameter); // Draw an ellipse with given parameters
}

Explanation:

  • void drawEllipse(float x, float y, float diameter): Defines a custom function that draws an ellipse based on parameters.
  • drawEllipse(400, 300, 100): Calls the function to draw an ellipse at the specified location and size.

Using functions helps to organize your code, making it more modular and reusable.

Tutorial 7: User Input

Objective: Learn how to handle user input such as mouse clicks and keyboard presses.

1. Handling Mouse Input


void setup() {
    size(800, 600);
    background(255);
}

void draw() {
    fill(255, 0, 0);
    if (mousePressed) {
        ellipse(mouseX, mouseY, 50, 50); // Draw an ellipse where the mouse is clicked
    }
}

Explanation:

  • if (mousePressed): Checks if the mouse is pressed.
  • ellipse(mouseX, mouseY, 50, 50): Draws an ellipse at the current mouse position when pressed.

Handling user input allows you to create interactive sketches that respond to user actions.

Tutorial 8: Creating Interactive Visuals

Objective: Combine previously learned concepts to create interactive and dynamic visuals.

1. Interactive Visualization


void setup() {
    size(800, 600);
}

void draw() {
    background(255);
    fill(map(mouseX, 0, width, 0, 255), 100, 150);
    ellipse(mouseX, height / 2, 100, 100); // Change the color of the ellipse based on mouse position
}

Explanation:

  • map(mouseX, 0, width, 0, 255): Maps the mouse X position to a color value, changing the ellipse color dynamically.

Combining concepts helps to create rich and interactive visual experiences, showcasing your understanding of Processing.
