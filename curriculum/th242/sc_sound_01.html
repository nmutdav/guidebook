Supercollider Tutorial Series - Sound 1

Tutorial 1: Basic Sound Playback

Objective: Learn how to play a sound file.


s.boot;  // Boot the Supercollider server

// Define a SynthDef to play a sample
(
SynthDef(\basicSound, {
    |out = 0, bufnum, amp = 0.5|
    var sound = PlayBuf.ar(1, bufnum, BufRateScale.kr(bufnum), doneAction: 2);
    sound = sound * amp;
    Out.ar(out, sound);
}).add;
)

// Load a sound file into a buffer
(
b = Buffer.read(s, "path/to/soundfile.wav"); // Replace with your sound file path
)

// Play the sound
(
Synth(\basicSound, [\bufnum, b]);
)

Explanation:

  • s.boot;: Starts the Supercollider server.
  • SynthDef: Defines a synthesis unit for playing a sample.
  • PlayBuf.ar: Plays the buffer content.
  • Buffer.read: Loads a sound file into a buffer.
  • Synth: Creates an instance of the defined SynthDef to play the sound.

This tutorial introduces basic sound playback functionality.

Tutorial 3: Applying ADSR Envelope

Objective: Learn how to apply an ADSR (Attack, Decay, Sustain, Release) envelope to a sound.


s.boot;  // Boot the Supercollider server

// Define a SynthDef with ADSR envelope
(
SynthDef(\adsrEnvelope, {
    |out = 0, bufnum, amp = 0.5, atk = 0.01, dec = 0.1, sus = 0.5, rel = 0.5|
    var env = EnvGen.kr(Env.perc(atk, dec, sus, rel), doneAction: 2);
    var sound = PlayBuf.ar(1, bufnum, BufRateScale.kr(bufnum), doneAction: 2);
    sound = sound * env * amp;
    Out.ar(out, sound);
}).add;
)

// Load a sound file into a buffer
(
b = Buffer.read(s, "path/to/soundfile.wav"); // Replace with your sound file path
)

// Play the sound with ADSR envelope
(
Synth(\adsrEnvelope, [\bufnum, b, \amp, 0.8, \atk, 0.02, \dec, 0.2, \sus, 0.7, \rel, 0.3]);
)

Explanation:

  • EnvGen.kr(Env.perc(...)): Creates an ADSR envelope with specified parameters.
  • atk, dec, sus, rel: Parameters for Attack, Decay, Sustain, and Release times.

This tutorial introduces applying an ADSR envelope to control the sound's amplitude over time.

Tutorial 5: Combining Effects

Objective: Learn how to combine multiple effects like volume control, ADSR envelope, and pitch shifting.


s.boot;  // Boot the Supercollider server

// Define a SynthDef combining volume, ADSR envelope, and pitch shifting
(
SynthDef(\combinedEffects, {
    |out = 0, bufnum, amp = 0.5, pitch = 1.0, atk = 0.01, dec = 0.1, sus = 0.5, rel = 0.5|
    var env = EnvGen.kr(Env.perc(atk, dec, sus, rel), doneAction: 2);
    var sound = PlayBuf.ar(1, bufnum, BufRateScale.kr(bufnum) * pitch, doneAction: 2);
    sound = sound * env * amp;
    Out.ar(out, sound);
}).add;
)

// Load a sound file into a buffer
(
b = Buffer.read(s, "path/to/soundfile.wav"); // Replace with your sound file path
)

// Play the sound with combined effects
(
Synth(\combinedEffects, [\bufnum, b, \amp, 0.8, \pitch, 1.5, \atk, 0.02, \dec, 0.2, \sus, 0.7, \rel, 0.3]);
)

Explanation:

  • This SynthDef combines volume control, ADSR envelope, and pitch shifting.

This tutorial demonstrates how to combine multiple sound effects for more complex sound design.
