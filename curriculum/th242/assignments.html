Assignments

In this class, we will use the Processing IDE to explore multimedia projects integrating audio and visual elements. Each assignment involves team-based collaboration with rotating members to help you build and refine your skills in creative coding. Individual contributions are required for each assignment to ensure personal accountability. We will use GitHub for collaboration, and each team will set up a repository and use the NMUTDAV GitHub (github.io) to host and showcase their projects.

Team Grouping and Roles

  • Team Formation: Teams will be formed for each assignment, lasting a minimum of three weeks. After this period, teams will be reorganized to ensure a mix of experiences and perspectives.
  • Roles:
    • Project Lead: Organizes team meetings, manages project assets, and oversees project progress.
    • Resource Curator: Gathers and organizes images, audio, and other assets relevant to the project.
    • Code Developer: Focuses on implementing the coding aspects of the project, including writing and debugging code.
    • Documenter: Maintains project documentation, including README files and assignment reflections.

Assignment 1: Basic Audio Visualization

  • Objective: Create a basic animation that visualizes audio playback.
  • Tasks:
    • Load an audio sample and play it using Processing.
    • Visualize the audio waveform or amplitude as a moving shape.
    • Experiment with different visualizations based on audio properties.
  • Team Collaboration: Rotate team roles and use GitHub for version control and project management.
  • Reflection: Submit a text file reflection on your experience and challenges.
  • GitHub Submission: Ensure your project is hosted on GitHub Pages. Submit the URL via email.
