Assignments

In this class, we will use the Processing IDE to explore multimedia projects that integrate audio and visual elements. Each assignment will involve team-based collaboration with rotating members, allowing you to build and refine your skills in creative coding. Individual contributions are required for each assignment to ensure personal accountability. We will use GitHub for collaboration, and each team will be provided with a repository to build upon, host, and showcase their projects.

Adjustment (09/10/2024): To better align our projects with sound and media design specifically for theatre, our upcoming assignments will focus on using scripts as inspiration for creating soundscapes and animations. I will provide a repository of public domain scripts along with guidance on which excerpts to use. In class, we will discuss strategies for interpreting these excerpts and how to effectively render sound and animations that complement theatrical productions. This adjustment is intended to help you develop foundational skills in interpreting a script, conceiving of a design, and executing that design, preparing you for future opportunities in designing for theatrical productions. While the assignments will remain similar to before, they will now feature more structure, guidance, and context, framed through the lens of theatre design.

Team Grouping and Roles

  • Team Formation: Teams will be formed for each assignment and reorganized periodically to ensure a mix of experiences and perspectives.
  • Roles:
    • Project Lead: Organizes team meetings, manages project assets, and oversees project progress.
    • Resource Curator: Gathers and organizes images, audio, and other assets relevant to the project.
    • Code Developer: Focuses on implementing the coding aspects of the project, including writing and debugging code.
    • Documenter: Maintains project documentation, including README files and assignment reflections.

Presentations

Presentations for each assignment should last between 10 to 15 minutes, followed by additional time for Q&A. Ensure your presentation effectively communicates your project's objectives, implementation, and outcomes.

Grading Breakdown

Each assignment will be evaluated based on the following criteria:

  • Project Implementation: 25%
  • Creativity and Innovation: 25%
  • User Interaction: 25%
  • Documentation and Quality: 25%

The final grade for each assignment will be based on a combination of these factors. Detailed feedback will be provided to help you improve and succeed in future assignments.

Assignments

Assignment 1: Audio-Visual Mood Board

  • Objective: Create a simple visual representation that reacts to an audio file, conveying a specific mood or emotion.
  • Tasks:
    • Choose a short audio clip that evokes a clear mood (e.g., calming, intense, joyful).
    • Create a visual animation that responds to the audio, such as changing colors or shapes based on the audio's amplitude.
    • Ensure the visuals match the mood of the audio clip, e.g., calming visuals for a soothing sound.
  • Reflection: Submit a brief text file reflecting on how the audio influenced your visual design and what challenges you encountered.
  • GitHub Submission: Ensure your project is hosted on GitHub with clear instructions on how to run your code.

Assignment 2: Interactive Soundscape

  • Objective: Develop an interactive visual environment where users can manipulate sounds and visuals in real-time, using a theatrical script excerpt as your primary source of inspiration.

    • Adjustment (09/10/2024): Each group will choose one excerpt from a provided selection of theatre scripts to develop their concept and design for this project. This approach will help you build foundational skills in interpreting a script, conceiving a design, and executing that design. Here are some ways to interpret this:

    • Script-Themed Interactions: Design interactive elements that reflect key themes or motifs from the chosen excerpt. For instance, if the excerpt involves a dramatic confrontation, create interactive audio-visual elements that capture the intensity and emotional stakes of that scene.
    • Character-Based Visuals and Sounds: Use specific characters from the script to inspire visual and auditory components. Develop distinct sounds and visual effects that represent the personalities or emotions of these characters, allowing users to engage with the environment in ways that reflect their traits.
    • Setting and Atmosphere: Translate the setting and atmosphere of the script excerpt into the interactive environment. For example, if the excerpt is set in a magical forest, incorporate visual elements like changing colors and sounds that evoke the enchanted nature of the setting.
  • Tasks:
    • Create a visual environment where different parts of the screen produce different sounds when interacted with, inspired by your chosen script excerpt.
    • Allow users to modify the visuals by interacting with the audio elements, such as changing colors or patterns, in a way that reflects the themes or atmosphere of the script.
    • Ensure that the interaction is intuitive, engaging, and clearly connected to the script excerpt.
  • Reflection: Write a reflection on how users interact with your project and how this interaction reflects elements from the script excerpt. Discuss how effectively your design captures the essence of the script and enhances the user experience.
  • GitHub Submission: Host your project on GitHub and include a README file with instructions for running the code, as well as an explanation of how the script excerpt influenced your design choices.

Assignment 3: Dynamic Visual Symphony

  • We will be using the script "Rossum's Universal Robots" for Assignment 3, which can be downloaded on the index page for this course. Please read the entire script for reference. Group 1 will use Act I, group 2 will use Act II, and Group 3 will use Act III as their focus for the assignment.

  • Objective: Create a visual system that translates various audio properties (e.g., pitch, tempo) into complex visual patterns and effects, using a theatrical script excerpt as the basis for your design concept.

    • Adjustment (09/10/2024): Each group will choose one excerpt from a provided selection of theatre scripts to guide the development of their dynamic visual system. This approach will help you integrate script elements into your visual design, enhancing your ability to interpret and visualize theatrical themes through audio-visual interactions. Here are some ways to incorporate the script into your design:

    • Script-Inspired Audio Properties: Select audio properties that align with the themes or emotions of the script excerpt. For instance, if the excerpt is dramatic, choose audio elements with varying intensities to reflect the script's emotional depth.
    • Thematic Visual Patterns: Design visual patterns and effects that echo the themes or settings of the script. For example, if the excerpt is set in a mystical realm, create visual effects that convey the magical or surreal aspects of that world.
    • Character and Scene Influence: Use the characteristics of specific characters or scenes from the script to influence how the visual effects respond to different audio properties. This could involve tailoring visual responses to reflect the personalities or actions described in the script.
  • Tasks:
    • Use an audio file with varied elements (e.g., music with different instruments and rhythms) as a basis for your visual system.
    • Develop visual effects that change dynamically in response to different audio properties like pitch and tempo, inspired by your chosen script excerpt.
    • Create a visual pattern that evolves as the audio plays, reflecting the audio's complexity and thematic elements from the script.
  • Reflection: Discuss how the audio properties influenced your visual design and how incorporating elements from the script excerpt impacted your project. Reflect on any challenges you faced in creating dynamic effects that align with the script's themes.
  • GitHub Submission: Ensure your project is properly documented on GitHub, including setup and running instructions, as well as an explanation of how the script excerpt influenced your visual design.

Final Assignment: Immersive Audio-Visual Experience

  • Objective: Collaborate on a comprehensive project that combines all techniques learned to create an immersive audio-visual experience, using a theatrical script excerpt as the foundation for your design.

    • Adjustment (09/10/2024): For this final project, each group will choose one excerpt from a provided selection of theatre scripts to guide the development of their immersive experience. This adjustment ensures that your project integrates theatrical themes and elements into a cohesive audio-visual presentation. Here are some ways to incorporate the script into your final project:

    • Script-Based Environment Design: Create an immersive environment that reflects the setting, themes, and mood of the chosen script excerpt. For example, if the excerpt is set in a dystopian future, design visual and audio elements that evoke that futuristic and possibly bleak atmosphere.
    • Character and Narrative Integration: Develop interactive elements that allow users to engage with the environment based on characters or narrative aspects from the script. This could involve user interactions that trigger specific audio-visual responses linked to key moments or characters in the script.
    • Thematic Audio-Visual Synchronization: Use advanced audio analysis to synchronize visual effects with the script’s thematic elements. For instance, if the script involves a dramatic tension, adjust audio and visual elements to reflect increasing intensity or changes in emotional tone.
  • Tasks:
    • Design an immersive environment that integrates advanced audio analysis and visual synchronization, inspired by your chosen script excerpt.
    • Include user interactions that offer a personalized experience, such as manipulating the environment through audio input that reflects elements from the script.
    • Ensure that the final project is polished, cohesive, and effectively captures the immersive experience aligned with the script's themes and narrative.
  • Reflection: Submit a detailed reflection on the overall project development process, including team dynamics, challenges faced, and how well the project achieved its immersive goals. Discuss how the script excerpt influenced your design choices and contributed to the immersive experience.
  • GitHub Submission: Host your final project on GitHub with comprehensive documentation, including a user guide, setup instructions, and an explanation of how the script excerpt shaped your project.
