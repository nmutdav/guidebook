Assignment Overview

In this class, we will use the Processing IDE exclusively to explore and develop multimedia projects involving both audio and visual elements. Each assignment will guide you through integrating audio and media into your Processing projects, helping you build and refine your skills in creative coding.

Template Code Snippets

Processing Template Code


// Basic Animation with Audio in Processing
import processing.sound.*;

SoundFile sample;
float angle = 0;
float radius = 100;
float speed = 0.05;

void setup() {
    size(800, 600);
    sample = new SoundFile(this, "path/to/sample.wav"); // Load audio sample
    sample.play(); // Start playback
}

void draw() {
    background(0);

    // Update color based on mouse position
    float r = map(mouseX, 0, width, 0, 255);
    float g = map(mouseY, 0, height, 0, 255);
    fill(r, g, 255 - r);

    // Draw a rotating ellipse
    float x = width / 2 + cos(angle) * radius;
    float y = height / 2 + sin(angle) * radius;
    ellipse(x, y, 50, 50);

    // Update angle for rotation
    angle += speed;

    // Draw a line from the center to the circle
    stroke(255);
    line(width / 2, height / 2, x, y);
}

// Adjust rotation speed with arrow keys
void keyPressed() {
    if (keyCode == LEFT) {
        speed -= 0.01;
    } else if (keyCode == RIGHT) {
        speed += 0.01;
    }
}

Assignments

Assignment 1: Basic Audio Visualization

  • Objective: Create a basic animation that visualizes audio playback.
  • Tasks:
    • Load an audio sample and play it using Processing.
    • Visualize the audio waveform or amplitude as a moving shape.
    • Experiment with different visualizations based on audio properties.
  • Reflection: Submit a written reflection on your experience. Discuss what you learned about audio visualization and the challenges faced.
  • Proposal for Next Assignment: Outline your approach for the next assignment, including initial ideas or concepts for integrating audio and visuals.
  • Class Presentation: Prepare a brief presentation showcasing your audio visualization. Demonstrate how the visual changes in response to the audio.

Assignment 2: Interactive Audio Feedback

  • Objective: Develop an interactive animation that responds to audio input or user actions.
  • Tasks:
    • Create animations that change based on audio playback or user interactions, such as mouse clicks or movements.
    • Implement features like adjusting audio volume or playback speed through user input.
  • Reflection: Submit a written reflection on your experience. Discuss how user interactions and audio influence the animation.
  • Proposal for Next Assignment: Outline your approach for the next assignment, including ideas for further enhancing interactivity with audio.
  • Class Presentation: Prepare a brief presentation of your interactive animation. Highlight how it responds to both audio and user inputs.

Assignment 3: Audio-Responsive Visual Effects

  • Objective: Create visual effects that respond dynamically to audio properties.
  • Tasks:
    • Analyze audio properties such as volume or frequency.
    • Use these properties to drive visual effects, such as changing colors, shapes, or animations based on the audio input.
  • Reflection: Submit a written reflection on your experience. Discuss how audio properties influenced your visual effects and any challenges encountered.
  • Proposal for Next Assignment: Outline your approach for the next assignment, including ideas for more complex audio-responsive features.
  • Class Presentation: Prepare a brief presentation demonstrating your audio-responsive visual effects. Showcase how different audio properties affect the visuals.
