Understanding Arrays in Processing for Theater Animations

Objective

By the end of this lecture, students will understand what arrays are and how to use them to create animations for theater productions in Processing.

1. What are Arrays?

An array is a way to store multiple values in a single variable. Think of it as a row of lockers where each locker can hold a piece of information. This allows you to manage a group of related data more efficiently.

Example:

        // Declare and initialize an array of colors
        color[] colors = {color(255, 0, 0), color(0, 255, 0), color(0, 0, 255)};

2. Why Use Arrays for Theater Animations?

In theater animations, you often have multiple characters, props, or scenes that need to change over time. Here’s why arrays are particularly useful:

2.1 Efficient Data Management

  • Group Related Data: Arrays allow you to store multiple values (e.g., positions, colors, velocities) in a single variable. This is essential when dealing with multiple animated objects, making the code cleaner and easier to manage.
  • Dynamic Access: You can use an index to access and manipulate any element in the array efficiently. This is particularly useful for updating positions or states of many objects simultaneously.

2.2 Iteration and Automation

  • Simplifying Loops: Arrays enable you to use loops to iterate through multiple elements. Instead of writing repetitive code for each object, a single loop can update or draw all objects, making the code shorter and easier to read.
  • Automated Changes: In animation, properties (like position and size) often change over time. Arrays make it simple to apply the same logic to many objects at once, automating their behavior systematically.

3. Creating a Simple Animation Using Arrays

3.1 Example: Moving Characters Across the Stage

Let’s create a simple animation where multiple characters move from left to right across the screen.

        // Declare an array for character positions
        float[] xPositions = new float[5]; // 5 characters
    
        void setup() {
            size(800, 400);
            // Initialize character positions
            for (int i = 0; i < xPositions.length; i++) {
                xPositions[i] = i * 150; // Space them out
            }
        }
    
        void draw() {
            background(255); // Clear the screen
    
            // Update positions and draw characters
            for (int i = 0; i < xPositions.length; i++) {
                xPositions[i] += 1; // Move character to the right
                fill(255, 0, 0); // Color for characters
                ellipse(xPositions[i], height / 2, 50, 50); // Draw character as a circle
            }
        }

4. Making It Interactive

We can make the animation more interesting by allowing characters to change direction when they reach the edge of the screen.

        // Declare an array for character positions and velocities
        float[] xPositions = new float[5];
        float[] velocities = new float[5];
    
        void setup() {
            size(800, 400);
            for (int i = 0; i < xPositions.length; i++) {
                xPositions[i] = i * 150;
                velocities[i] = random(1, 5); // Random speeds
            }
        }
    
        void draw() {
            background(255);
            for (int i = 0; i < xPositions.length; i++) {
                xPositions[i] += velocities[i];
                if (xPositions[i] > width || xPositions[i] < 0) {
                    velocities[i] *= -1; // Change direction
                }
                fill(255, 0, 0);
                ellipse(xPositions[i], height / 2, 50, 50);
            }
        }

5. Conclusion

Arrays are powerful tools for managing multiple elements in your animations. By using arrays, you can efficiently control the positions and behaviors of characters in your theater projects.
