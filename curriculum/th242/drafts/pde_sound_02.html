Processing Tutorial Series - Sound 2

Tutorial 1: Basic Sound Manipulation

Objective: Learn how to control sound playback speed.


import processing.sound.*;

SoundFile mySound;

void setup() {
    size(800, 600);
    mySound = new SoundFile(this, "soundfile.mp3");
    mySound.play();
}

void draw() {
    background(255);
}

void keyPressed() {
    if (key == 'a') {
        mySound.rate(0.5); // Slow down playback
    } else if (key == 's') {
        mySound.rate(1.5); // Speed up playback
    }
}

Explanation:

  • mySound.rate(float rate);: Controls the playback speed of the sound.
  • keyPressed(): Adjusts the playback speed based on key presses.

This tutorial introduces basic sound manipulation through playback speed control.

Tutorial 2: Sound Effects with Filters

Objective: Apply a low-pass filter to a sound file.


import processing.sound.*;

SoundFile mySound;
Filter lowPassFilter;

void setup() {
    size(800, 600);
    mySound = new SoundFile(this, "soundfile.mp3");
    mySound.play();
    
    lowPassFilter = new Filter(this);
    lowPassFilter.set(0.5); // Set filter frequency
    lowPassFilter.input(mySound);
}

void draw() {
    background(255);
}

Explanation:

  • Filter lowPassFilter;: Creates a filter object for audio processing.
  • lowPassFilter.set(float frequency);: Sets the filter frequency to affect the sound.
  • lowPassFilter.input(mySound);: Applies the filter to the sound file.

This tutorial demonstrates applying a low-pass filter to modify sound characteristics.

Tutorial 4: Dynamic Sound Control

Objective: Adjust sound parameters dynamically using sliders.


import processing.sound.*;
import controlP5.*;

SoundFile mySound;
ControlP5 cp5;
Slider volumeSlider;

void setup() {
    size(800, 600);
    mySound = new SoundFile(this, "soundfile.mp3");
    mySound.play();
    
    cp5 = new ControlP5(this);
    volumeSlider = cp5.addSlider("volume")
                      .setPosition(50, 50)
                      .setSize(200, 20)
                      .setRange(0, 1)
                      .setValue(0.5);
}

void draw() {
    background(255);
    mySound.amp(volumeSlider.getValue());
}

Explanation:

  • ControlP5 cp5;: Creates a ControlP5 object for GUI controls.
  • Slider volumeSlider;: Adds a slider to control volume.
  • volumeSlider.getValue();: Retrieves the current value from the slider and adjusts the sound volume accordingly.

This tutorial demonstrates how to use sliders for real-time control of sound parameters.

Tutorial 5: Sound Analysis Visualization

Objective: Visualize the frequency spectrum of a sound file.


import processing.sound.*;
import processing.core.PApplet;

SoundFile mySound;
FFT fft;

void setup() {
    size(800, 600);
    mySound = new SoundFile(this, "soundfile.mp3");
    mySound.play();
    
    fft = new FFT(this, 512); // Set FFT size
    fft.input(mySound);
}

void draw() {
    background(255);
    
    fft.analyze(); // Perform FFT analysis
    stroke(0);
    noFill();
    
    // Draw the frequency spectrum
    for (int i = 0; i < fft.specSize(); i++) {
        float x = map(i, 0, fft.specSize(), 0, width);
        float y = map(fft.getBand(i), 0, 1, height, 0);
        ellipse(x, y, 5, 5);
    }
}

Explanation:

  • FFT fft;: Creates an FFT object for frequency analysis.
  • fft.analyze();: Performs an FFT analysis on the sound.
  • ellipse(x, y, 5, 5);: Visualizes frequency data as ellipses.

This tutorial covers visualizing the frequency spectrum of sound using FFT analysis.

Tutorial 6: Creating a Sound Synthesizer

Objective: Build a simple sound synthesizer with customizable oscillators.


import processing.sound.*;

Oscillator osc;
float frequency = 440; // Frequency in Hz

void setup() {
    size(800, 600);
    osc = new Oscillator(this);
    osc.setFrequency(frequency);
    osc.play();
}

void draw() {
    background(255);
}

void keyPressed() {
    if (key == 'a') {
        frequency += 50; // Increase frequency
        osc.setFrequency(frequency);
    } else if (key == 's') {
        frequency -= 50; // Decrease frequency
        osc.setFrequency(frequency);
    }
}

Explanation:

  • Oscillator osc;: Creates an oscillator for generating sound.
  • osc.setFrequency(float freq);: Sets the frequency of the oscillator.
  • keyPressed(): Adjusts the frequency based on key presses.

This tutorial demonstrates building a simple synthesizer with adjustable frequency.

Tutorial 7: Interactive Sound Environment

Objective: Create an interactive sound environment based on mouse input.


import processing.sound.*;

SoundFile mySound;
float pitch = 1.0;

void setup() {
    size(800, 600);
    mySound = new SoundFile(this, "soundfile.mp3");
    mySound.play();
}

void draw() {
    background(255);
    pitch = map(mouseX, 0, width, 0.5, 2.0); // Map mouse position to pitch
    mySound.rate(pitch);
}

Explanation:

  • map(value, start1, stop1, start2, stop2);: Maps mouse position to pitch value.
  • mySound.rate(float rate);: Adjusts the pitch of the sound based on mouse position.

This tutorial covers creating an interactive sound environment using mouse input to control pitch.
