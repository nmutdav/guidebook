Adding Humor to Your Project

Due: Class 7

Objective: Integrate humorous elements into your Processing sketch.

Step-by-Step Instructions:

  1. Review Your Current Sketch:

    Open the Processing sketch from Assignment 3 where you created surreal visuals and audio effects. This will be the base for adding humorous elements.

  2. Add Humorous Visual Elements:

    Modify your sketch to include playful or funny visuals. For example, use cartoonish shapes or animations. Hereâ€™s a basic example:

    
void setup() {
  size(600, 600);
  background(255);
}

void draw() {
  background(255);
  fill(255, 0, 0);
  ellipse(mouseX, mouseY, 100, 100); // Draw a red circle where the mouse is
  fill(0);
  textSize(32);
  text("Funny Face", mouseX - 50, mouseY + 10); // Display text near the circle
}

    This code draws a red circle following the mouse with humorous text. Customize it to fit your style.

  3. Incorporate Humorous Audio Effects:

    Add funny sound effects to your project. You can use sound effects like laughs or silly noises. Example code using Minim:

    
import ddf.minim.*;

Minim minim;
AudioPlayer player;

void setup() {
  size(600, 600);
  minim = new Minim(this);
  player = minim.loadFile("funny_sound.mp3"); // Replace with your sound file
  player.play();
}

void draw() {
  // Example: play sound when the mouse is pressed
  if (mousePressed) {
    player.rewind();
    player.play();
  }
}

    Replace `"funny_sound.mp3"` with a humorous sound effect of your choice.

  4. Test and Refine:

    Run your sketch to ensure the humorous elements are working as expected. Make adjustments to enhance the comedic effect.
