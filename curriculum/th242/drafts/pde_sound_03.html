Processing Tutorial Series - Sound 3

Tutorial 1: Advanced Sound Synthesis

Objective: Create a sound synthesizer with multiple oscillators and modulation.


import processing.sound.*;

Oscillator osc1, osc2;
float frequency1 = 440; // Base frequency for osc1
float frequency2 = 880; // Base frequency for osc2

void setup() {
    size(800, 600);
    osc1 = new Oscillator(this);
    osc2 = new Oscillator(this);
    
    osc1.setFrequency(frequency1);
    osc2.setFrequency(frequency2);
    
    osc1.play();
    osc2.play();
}

void draw() {
    background(255);
    float modFreq = map(mouseX, 0, width, 1, 10); // Modulation frequency based on mouse position
    osc2.setFrequency(frequency2 + sin(frameCount * 0.05) * 50); // Frequency modulation
}

Explanation:

  • Oscillator osc1, osc2;: Creates two oscillators for sound synthesis.
  • osc2.setFrequency(float freq);: Modulates the frequency of the second oscillator.
  • map(value, start1, stop1, start2, stop2);: Maps mouse position to modulation frequency.

This tutorial involves creating a complex synthesizer with modulation effects.

Tutorial 2: Real-Time Audio Filtering

Objective: Apply a band-pass filter to isolate specific frequencies.


import processing.sound.*;

SoundFile mySound;
Filter bandPassFilter;

void setup() {
    size(800, 600);
    mySound = new SoundFile(this, "soundfile.mp3");
    mySound.play();
    
    bandPassFilter = new Filter(this);
    bandPassFilter.set(1000, 0.5); // Set filter frequency and resonance
    bandPassFilter.input(mySound);
}

void draw() {
    background(255);
}

Explanation:

  • Filter bandPassFilter;: Creates a band-pass filter for isolating frequencies.
  • bandPassFilter.set(float frequency, float resonance);: Sets the filter's frequency and resonance.
  • bandPassFilter.input(SoundFile sound);: Applies the filter to the sound file.

This tutorial demonstrates filtering specific frequencies from a sound using a band-pass filter.

Tutorial 3: Complex Sound Effects

Objective: Combine multiple effects like reverb, delay, and distortion.


import processing.sound.*;

SoundFile mySound;
Reverb reverb;
Delay delay;
Distortion distortion;

void setup() {
    size(800, 600);
    mySound = new SoundFile(this, "soundfile.mp3");
    mySound.play();
    
    reverb = new Reverb(this);
    delay = new Delay(this);
    distortion = new Distortion(this);
    
    reverb.process(mySound, 0.5); // Apply reverb
    delay.process(mySound, 500); // Apply delay
    distortion.process(mySound, 0.3); // Apply distortion
}

void draw() {
    background(255);
}

Explanation:

  • Reverb reverb;: Creates a reverb effect object.
  • Delay delay;: Creates a delay effect object.
  • Distortion distortion;: Creates a distortion effect object.
  • process(SoundFile sound, float amount);: Applies effects to the sound file.

This tutorial combines multiple effects to create complex sound textures.

Tutorial 4: Audio Reactive Visualization

Objective: Create visual effects that respond to audio input.


import processing.sound.*;

SoundFile mySound;
FFT fft;

void setup() {
    size(800, 600);
    mySound = new SoundFile(this, "soundfile.mp3");
    mySound.play();
    
    fft = new FFT(this, 512);
    fft.input(mySound);
}

void draw() {
    background(255);
    
    fft.analyze();
    
    stroke(0);
    noFill();
    
    for (int i = 0; i < fft.specSize(); i++) {
        float x = map(i, 0, fft.specSize(), 0, width);
        float y = map(fft.getBand(i), 0, 1, height, 0);
        ellipse(x, y, 5, 5);
    }
}

Explanation:

  • FFT fft;: Creates an FFT object for frequency analysis.
  • fft.analyze();: Analyzes the sound file and updates the visual representation.
  • ellipse(x, y, 5, 5);: Visualizes frequency data with ellipses.

This tutorial creates visual effects that respond to audio input using FFT analysis.

Tutorial 5: Audio Sampling and Playback

Objective: Sample and playback audio segments.


import processing.sound.*;

SoundFile mySound;
AudioSample sample;

void setup() {
    size(800, 600);
    mySound = new SoundFile(this, "soundfile.mp3");
    mySound.play();
    
    sample = new AudioSample(this, "soundfile.mp3");
}

void draw() {
    background(255);
}

void keyPressed() {
    if (key == 's') {
        sample.play(); // Play sample
    }
}

Explanation:

  • AudioSample sample;: Creates an audio sample object for playback.
  • sample.play();: Plays the audio sample when a key is pressed.

This tutorial covers sampling and playback of audio segments.

Tutorial 8: Creating Custom Soundscapes

Objective: Design complex soundscapes using multiple sound files and effects.


import processing.sound.*;

SoundFile[] sounds = new SoundFile[3];
Reverb[] reverbs = new Reverb[3];

void setup() {
    size(800, 600);
    sounds[0] = new SoundFile(this, "sound1.mp3");
    sounds[1] = new SoundFile(this, "sound2.mp3");
    sounds[2] = new SoundFile(this, "sound3.mp3");
    
    for (int i = 0; i < 3; i++) {
        sounds[i].play();
        reverbs[i] = new Reverb(this);
        reverbs[i].process(sounds[i], 0.3); // Apply reverb to each sound
    }
}

void draw() {
    background(255);
}

Explanation:

  • SoundFile[] sounds = new SoundFile[3];: Creates an array of sound files.
  • Reverb[] reverbs = new Reverb[3];: Creates an array of reverb effect objects.
  • for (int i = 0; i < 3; i++) { ... }: Applies reverb to each sound file.

This tutorial demonstrates creating complex soundscapes by combining multiple sound files with effects.
