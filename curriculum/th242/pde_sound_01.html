Processing Tutorial Series - Sound 1

Tutorial 1: Basic Sound Playback

Objective: Learn how to play a sound file.


import processing.sound.*;

SoundFile mySound;

void setup() {
    size(800, 600);
    mySound = new SoundFile(this, "soundfile.mp3");
    mySound.play();
}

void draw() {
    background(255);
}

Explanation:

  • SoundFile mySound;: Creates a SoundFile object to manage sound playback.
  • mySound.play();: Starts playing the sound file.

This tutorial introduces basic sound playback functionality.

Tutorial 2: Toggle Playback

Objective: Learn how to toggle sound playback on and off with mouse clicks.


import processing.sound.*;

SoundFile mySound;
boolean isPlaying = false;

void setup() {
    size(800, 600);
    mySound = new SoundFile(this, "soundfile.mp3");
}

void draw() {
    background(255);
}

void mousePressed() {
    if (isPlaying) {
        mySound.pause();
    } else {
        mySound.play();
    }
    isPlaying = !isPlaying;
}

Explanation:

  • boolean isPlaying = false;: Tracks whether the sound is currently playing.
  • mousePressed(): Toggles the playback state of the sound when the mouse is pressed.

This tutorial demonstrates how to control sound playback using mouse clicks.

Tutorial 5: Visualizing Volume

Objective: Learn how to visualize the volume level of sound playback.


import processing.sound.*;

SoundFile mySound;
Amplitude amp;

void setup() {
    size(800, 600);
    
    // Initialize SoundFile and Amplitude objects
    mySound = new SoundFile(this, "soundfile.mp3");
    amp = new Amplitude(this);
    
    // Play the sound
    mySound.play();
    
    // Attach the Amplitude object to the SoundFile
    amp.input(mySound);
}

void draw() {
    background(255);
    
    // Get the current volume level
    float volume = amp.analyze(); // Correct function to get amplitude
    
    // Use volume level for visual feedback or other purposes
    fill(0);
    ellipse(width / 2, height / 2, volume * 200, volume * 200); // Example visualization
}

Explanation:

  • Amplitude amp;: Creates an Amplitude object to measure the sound's amplitude (volume).
  • float volume = amp.analyze();: Retrieves the current volume level of the sound.

This tutorial introduces visualizing the amplitude of sound playback.

Tutorial 6: Applying Reverb Effect

Objective: Learn how to apply a reverb effect to a sound playback.


import processing.sound.*;

SoundFile mySound;
Reverb reverb;
float volume = 0.5;  // Volume level for both original and reverb

void setup() {
    size(800, 600);
    background(255);
    fill(0);
    textSize(32);
    textAlign(CENTER, CENTER);
    text("Press SPACEBAR to play sound", width / 2, height / 2);
    
    // Load sound file
    mySound = new SoundFile(this, "soundfile.mp3"); // Replace with your sound file
    
    // Initialize Reverb
    reverb = new Reverb(this);
    
    // Set reverb parameters
    float reverbSize = 1.0;     // Size of the reverb (0.0 to 1.0)
    float reverbDamping = 1.0;  // Damping of the reverb (0.0 to 1.0)
    float reverbWidth = 0.5;    // Width of the reverb (0.0 to 1.0)
    
    // Apply reverb effect
    reverb.set(reverbSize, reverbDamping, reverbWidth);
}

void draw() {
    // Instructions are already displayed in setup()
}

void keyPressed() {
    if (key == ' ') {
        // Play the original sound
        mySound.play();
        
        // Play the reverb-processed sound
        SoundFile reverbSound = new SoundFile(this, "soundfile.mp3");
        reverbSound.play();
        
        // Apply reverb to the sound
        reverb.process(reverbSound);
        
        // Adjust volume for both original and reverb sounds
        mySound.amp(volume);
        reverbSound.amp(volume);
    }
}

Explanation:

  • Reverb reverb;: Creates a Reverb object to apply the reverb effect.
  • reverb.set(reverbSize, reverbDamping, reverbWidth);: Sets the reverb effect parameters.
  • reverb.process(reverbSound);: Applies the reverb effect to the sound.
  • mySound.amp(volume); and reverbSound.amp(volume);: Adjust the volume level for both original and reverb sounds.

This tutorial demonstrates applying a reverb effect to sound playback.

Tutorial 7: Combining All Skills

Objective: Learn how to apply reverb, visualize volume, and play multiple sounds simultaneously.


import processing.sound.*;

SoundFile sound1;
SoundFile sound2;
Amplitude amp;
Reverb reverb;
float volume = 0.5;  // Volume level

void setup() {
    size(800, 600);
    background(255);
    fill(0);
    textSize(32);
    textAlign(CENTER, CENTER);
    text("Press SPACEBAR to play sounds", width / 2, height / 2);
    
    // Load sound files
    sound1 = new SoundFile(this, "soundfile1.mp3"); // Replace with your first sound file
    sound2 = new SoundFile(this, "soundfile2.mp3"); // Replace with your second sound file
    
    // Initialize Reverb
    reverb = new Reverb(this);
    
    // Set reverb parameters
    float reverbSize = 1.0;
    float reverbDamping = 1.0;
    float reverbWidth = 0.5;
    reverb.set(reverbSize, reverbDamping, reverbWidth);
    
    // Initialize Amplitude
    amp = new Amplitude(this);
    amp.input(sound1); // Attach the Amplitude object to the first sound
}

void draw() {
    background(255);
    
    // Get the current volume level for visualization
    float volumeLevel = amp.analyze();
    
    // Visualize the volume level
    fill(0, 100, 255, 150); // Semi-transparent blue
    float ellipseSize = volumeLevel * 400;
    ellipse(width / 2, height / 2, ellipseSize, ellipseSize);
}

void keyPressed() {
    if (key == ' ') {
        // Play both sounds
        sound1.play();
        sound2.play();
        
        // Apply reverb to both sounds
        reverb.process(sound1);
        reverb.process(sound2);
        
        // Adjust volume for both sounds
        sound1.amp(volume);
        sound2.amp(volume);
    }
}

Explanation:

  • SoundFile sound1; and SoundFile sound2;: Creates two SoundFile objects for managing playback of two separate sound files.
  • Amplitude amp;: Measures the amplitude (volume) of the first sound.
  • Reverb reverb;: Applies reverb effect to both sounds.
  • float volumeLevel = amp.analyze();: Retrieves the current volume level of the first sound for visualization.
  • ellipse(width / 2, height / 2, ellipseSize, ellipseSize);: Visualizes the volume level as an ellipse.
  • sound1.play(); and sound2.play();: Plays both sound files simultaneously when the SPACEBAR is pressed.
  • reverb.process(sound1); and reverb.process(sound2);: Applies reverb to both sound files.
  • sound1.amp(volume); and sound2.amp(volume);: Adjusts the volume for both sounds.

This final tutorial combines all the skills learned: playing multiple sounds, applying reverb, visualizing volume, and adjusting volume levels.
