TH-242: Media and Sound Design - Assignment 3

Surreal Visual and Audio Integration

Due: Class 5

Objective: Create a Processing sketch with surreal visual and audio effects.

Step-by-Step Instructions:

  1. Start with Your Existing Sketch:

    Open the Processing sketch from Assignment 2 where you integrated visuals and audio controls. This will be the base for adding surreal effects.

  2. Add Surreal Visuals:

    Modify your sketch to include unusual shapes, colors, and patterns. Use code like this for creating surreal visuals:

    
void setup() {
  size(600, 600);
  background(0);
}

void draw() {
  background(0);
  noStroke();
  fill(random(255), random(255), random(255), 100);
  ellipse(random(width), random(height), random(50, 150), random(50, 150));
  fill(random(255), random(255), random(255), 100);
  rect(random(width), random(height), random(50, 150), random(50, 150));
}

    This code creates random colorful ellipses and rectangles, giving a surreal look. You can modify shapes and colors to fit your vision.

  3. Incorporate Surreal Audio Effects:

    Modify your audio playback to include effects like pitch shifting or reverb. Use the Minim libraryâ€™s audio effects for this. Example code:

    
import ddf.minim.*;
import ddf.minim.effects.*;

Minim minim;
AudioPlayer player;
Reverb reverb;

void setup() {
  size(600, 600);
  minim = new Minim(this);
  player = minim.loadFile("your_audio_file.mp3"); // Replace with your file name
  player.play();
  reverb = new Reverb();
  reverb.setDecay(0.9);
  reverb.setDamping(0.7);
}

void draw() {
  // Apply reverb to audio
  reverb.process(player.mix);
}

    Replace `"your_audio_file.mp3"` with your audio file name. Adjust the reverb settings to create a more surreal sound.

  4. Test and Refine:

    Run your sketch and ensure that both the visuals and audio create a surreal experience. Adjust parameters as needed to enhance the surreal effect.

  5. Submit:

    • Submit your updated Processing sketch file (.pde) showcasing surreal visuals and audio effects.
    • Include a brief description of the surreal techniques used in both visuals and audio, and any challenges faced.
