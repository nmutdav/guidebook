TH-242: Media and Sound Design - Assignment 1

Basic Setup and Exploration

Due: Class 3

Objective: Get familiar with the Processing IDE and play an audio file using the Minim library.

Step-by-Step Instructions:

  1. Install Processing:

    • Go to the Processing website and download the Processing IDE for your operating system.
    • Follow the installation instructions for your operating system (Windows, macOS, or Linux).

  2. Install the Minim Library:

    • Open Processing.
    • Click on "Sketch" > "Import Library" > "Add Library".
    • Search for "Minim" and click "Install".

  3. Create Your First Sketch:

    In the Processing code editor, type the following code to draw a simple circle on the screen:

    
void setup() {
  size(400, 400);         // Set window size
  background(255);        // Set background color to white
  fill(0);                // Set circle color to black
  ellipse(width/2, height/2, 200, 200);  // Draw a circle in the center
}

  4. Play an Audio File:

    • Download a sample `.mp3` file and save it in your Processing sketch folder (same directory as your `.pde` file).
    • Add this code to your sketch to load and play the audio file:
      • 
import ddf.minim.*;

Minim minim;
AudioPlayer player;

void setup() {
  size(400, 400);
  minim = new Minim(this);
  player = minim.loadFile("your_audio_file.mp3"); // replace with your file name
  player.play();
}

void draw() {
  // Optional: Add some visuals that react to the music
}
    • Make sure the `.mp3` file is in the same folder as your Processing sketch.

  5. Reflection:

    • Write a brief reflection on your experience setting up Processing and Minim. What challenges did you face? How did you resolve them?

  6. Submit:

    • Submit your Processing sketch along with your reflection to the course portal.
