TH-242: Media and Sound Design - Assignment 5

Incorporating Nature into Your Project

Due: Class 8

Objective: Add elements of nature to your Processing sketch while maintaining the previous themes of humor and surrealism.

Step-by-Step Instructions:

  1. Review Your Current Sketch:

    Open the Processing sketch from Assignment 4 where you added humorous elements. This will be the base for integrating nature-themed elements.

  2. Integrate Nature-Themed Visuals:

    Add visual elements that represent nature, such as trees, water, or animals. For example:

    
void setup() {
  size(600, 600);
  background(255);
}

void draw() {
  background(255);
  fill(0, 255, 0);
  rect(100, 500, 200, 50); // Draw a tree trunk
  fill(0, 128, 0);
  ellipse(200, 450, 150, 150); // Draw the tree canopy
}

    This code adds a basic tree to the sketch. Customize the visuals to match your project's style.

  3. Incorporate Nature Sounds:

    Add sounds that evoke nature, such as birds chirping or water flowing. Example code using Minim:

    
import ddf.minim.*;

Minim minim;
AudioPlayer natureSound;

void setup() {
  size(600, 600);
  minim = new Minim(this);
  natureSound = minim.loadFile("nature_sound.mp3"); // Replace with your nature sound file
  natureSound.loop(); // Loop the nature sound
}

void draw() {
  // Example: adjust volume based on user input
  if (mousePressed) {
    natureSound.setGain(10); // Increase volume when mouse is pressed
  }
}

    Replace `"nature_sound.mp3"` with a nature-themed sound effect of your choice.

  4. Ensure Consistency:

    Make sure that the nature elements blend well with the surreal and humorous aspects of your project. Adjust visuals and sounds as needed.

  5. Submit:

    • Submit your updated Processing sketch file (.pde) that includes nature-themed visuals and sounds.
    • Provide a brief summary explaining how the nature elements integrate with the existing themes of humor and surrealism.
