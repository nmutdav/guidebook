TH-242: Media and Sound Design - Assignment 2

Integrating Visuals and Audio

Due: Class 5

Objective: Combine visual elements with basic audio controls in Processing.

Step-by-Step Instructions:

  1. Extend Your Previous Sketch:

    Open the Processing sketch from Assignment 1 where you created simple shapes and played audio. You will modify this sketch to include interactive elements.

  2. Add Interactive Shapes:

    Modify your sketch to include shapes that change color or size when you interact with them. Use the following code as a starting point:

    
boolean isMouseOver = false;

void setup() {
  size(400, 400);
  noStroke();
}

void draw() {
  background(255);
  if (isMouseOver) {
    fill(255, 0, 0);  // Red color
  } else {
    fill(0, 0, 255);  // Blue color
  }
  ellipse(mouseX, mouseY, 100, 100);  // Draw a circle at mouse position
}

void mouseMoved() {
  isMouseOver = dist(mouseX, mouseY, width / 2, height / 2) < 50;
}

  3. Implement Basic Audio Controls:

    Add functionality to control audio playback using keyboard or mouse inputs. Use the following code snippet to control playback:

    
import ddf.minim.*;

Minim minim;
AudioPlayer player;

void setup() {
  size(400, 400);
  minim = new Minim(this);
  player = minim.loadFile("your_audio_file.mp3"); // Replace with your file name
  player.play();
}

void keyPressed() {
  if (key == ' ') {
    if (player.isPlaying()) {
      player.pause();
    } else {
      player.play();
    }
  }
}

void mousePressed() {
  if (player.isPlaying()) {
    player.pause();
  } else {
    player.play();
  }
}

    Replace `"your_audio_file.mp3"` with the name of your audio file. The spacebar or mouse click will toggle play/pause.

  4. Test and Refine:

    Run your sketch and interact with the visuals and audio controls. Make sure that the shapes respond to mouse movements and the audio can be controlled as expected.

  5. Submit:

    • Submit your updated Processing sketch file (.pde) demonstrating interactive visuals and audio controls.
    • Include a brief description of how you integrated visuals with audio and any challenges you encountered.
