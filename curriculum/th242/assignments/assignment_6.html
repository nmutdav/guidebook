Enhancing Interactivity

Due: Class 9

Objective: Improve the interactivity of your Processing sketch by adding more user controls or responses.

Step-by-Step Instructions:

  1. Open Your Current Sketch:

    Open the Processing sketch from Assignment 5 where you integrated nature elements. This will be the base for enhancing interactivity.

  2. Add Interactive Elements:

    Include user controls that respond to input such as mouse clicks, keyboard presses, or other interactions. Example code for adding mouse control:

    
void setup() {
  size(600, 600);
}

void draw() {
  background(255);
  if (mousePressed) {
    fill(255, 0, 0);
    ellipse(mouseX, mouseY, 50, 50); // Draw a red circle where the mouse is clicked
  }
}

    This code draws a red circle at the mouse location when clicked. Customize this to fit your project.

  3. Implement Additional User Controls:

    Add more interactivity, such as keyboard controls to change colors or adjust sounds. Example code for changing colors with keyboard keys:

    
color bgColor = color(255);

void setup() {
  size(600, 600);
}

void draw() {
  background(bgColor);
}

void keyPressed() {
  if (key == 'r') {
    bgColor = color(255, 0, 0); // Change background to red
  } else if (key == 'g') {
    bgColor = color(0, 255, 0); // Change background to green
  }
}

    Test your sketch to ensure all interactive features work smoothly. Refine your code to improve user experience.
