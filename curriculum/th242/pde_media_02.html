Tutorial 1: Advanced Drawing Techniques

Objective: Explore advanced techniques for drawing complex shapes and patterns.

1. Drawing Complex Shapes


void setup() {
    size(800, 600);
    background(255);
}

void draw() {
    background(255);
    noFill(); // Disable filling shapes
    
    stroke(0, 100, 255); // Set stroke color
    strokeWeight(2); // Set stroke weight
    beginShape();
    vertex(100, 100);
    vertex(150, 50);
    vertex(200, 100);
    endShape(CLOSE); // Draw a complex shape
}

Explanation:

  • noFill(): Disables filling shapes, useful for drawing outlines.
  • stroke(0, 100, 255): Sets the color for the outline of shapes.
  • beginShape() and endShape(CLOSE): Define a custom shape by connecting vertices.

Advanced drawing techniques enable the creation of intricate patterns and designs.

Tutorial 2: Working with Images

Objective: Learn how to load and manipulate images in Processing.

1. Loading and Displaying Images


PImage img;

void setup() {
    size(800, 600);
    img = loadImage("path/to/your/image.jpg"); // Load an image file
}

void draw() {
    background(255);
    image(img, 0, 0); // Display the image
}

Explanation:

  • PImage img: Declares a variable for the image.
  • loadImage("path/to/your/image.jpg"): Loads an image from a specified path.
  • image(img, 0, 0): Draws the image at the specified position.

Loading and manipulating images allows for incorporating external visual elements into your sketches.

Tutorial 3: Interactivity with Keyboard

Objective: Implement keyboard controls to interact with your sketches.

1. Handling Keyboard Input


void setup() {
    size(800, 600);
}

void draw() {
    background(255);
    fill(255, 0, 0);
    if (keyPressed) {
        if (key == 'a') {
            ellipse(400, 300, 100, 100); // Draw ellipse if 'a' is pressed
        }
    }
}

Explanation:

  • keyPressed: Checks if any key is pressed.
  • key == 'a': Checks if the specific key ('a') is pressed.

Handling keyboard input allows for creating more interactive and responsive sketches.

Tutorial 4: Advanced Animation Techniques

Objective: Create more sophisticated animations by manipulating multiple parameters.

1. Creating Advanced Animations


float x = 0;
float y = 0;
float speedX = 2;
float speedY = 1;

void setup() {
    size(800, 600);
}

void draw() {
    background(255);
    fill(0, 100, 255);
    ellipse(x, y, 50, 50);
    x += speedX;
    y += speedY;
    
    if (x > width || x < 0) {
        speedX *= -1; // Reverse direction on horizontal edges
    }
    if (y > height || y < 0) {
        speedY *= -1; // Reverse direction on vertical edges
    }
}

Explanation:

  • speedX and speedY: Variables controlling the speed and direction of the animation.
  • x += speedX and y += speedY: Update the position of the ellipse.
  • if (x > width || x < 0): Reverse the direction if the ellipse hits the edges.

Advanced animation techniques enable more complex and dynamic movements within your sketches.

Tutorial 5: Using External Libraries

Objective: Learn to use external libraries to extend Processing's functionality.

1. Incorporating External Libraries


import processing.video.*;

Movie myMovie;

void setup() {
    size(800, 600);
    myMovie = new Movie(this, "path/to/your/video.mp4"); // Load a video file
    myMovie.loop(); // Play the video in a loop
}

void draw() {
    image(myMovie, 0, 0); // Display the video
}

Explanation:

  • import processing.video.*: Imports the video library.
  • Movie myMovie: Declares a variable for the video.
  • new Movie(this, "path/to/your/video.mp4"): Loads a video file.

Using external libraries allows for expanding the capabilities of Processing to include additional features and media types.

Tutorial 6: Creating User Interfaces

Objective: Build user interfaces with interactive components in Processing.

1. Building a Simple Interface


import controlP5.*;

ControlP5 cp5;

void setup() {
    size(800, 600);
    cp5 = new ControlP5(this);
    
    cp5.addButton("Click Me")
       .setPosition(100, 100)
       .setSize(200, 50)
       .onClick(new CallbackListener() {
           public void controlEvent(CallbackEvent theEvent) {
               println("Button Clicked!");
           }
       });
}

void draw() {
    background(255);
}

Explanation:

  • import controlP5.*: Imports the ControlP5 library for UI components.
  • ControlP5 cp5: Declares a ControlP5 instance.
  • addButton("Click Me"): Creates a button labeled "Click Me."
  • onClick(new CallbackListener()): Defines an action when the button is clicked.

Creating user interfaces enables interaction with your sketches through graphical controls.
