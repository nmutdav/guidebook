Processing Sound Design Tutorial Series 3

Tutorial 1: Creating a Granular Synthesizer

Objective: Design a granular synthesizer to manipulate sound by breaking it into grains.

1. Implementing Granular Synthesis:


import processing.sound.*;

SoundFile mySound;
Granular granular;

void setup() {
    size(800, 600);
    mySound = new SoundFile(this, "soundfile.wav");
    granular = new Granular(this);
    granular.process(mySound, 100, 0.5); // Process the sound with 100 grains and 0.5 density
    mySound.play();
}

void draw() {
    background(255);
}

Explanation:

  • Granular granular;: Creates a granular synthesizer object.
  • granular.process(mySound, 100, 0.5);: Processes the sound file with granular synthesis.

This tutorial introduces granular synthesis, allowing you to manipulate audio by dividing it into grains.

Tutorial 2: Building a Waveform Visualizer

Objective: Visualize the waveform of an audio file in real-time.

1. Displaying the Waveform:


import processing.sound.*;

SoundFile mySound;
Amplitude amplitude;
float[] waveform;

void setup() {
    size(800, 600);
    mySound = new SoundFile(this, "soundfile.wav");
    mySound.play();
    amplitude = new Amplitude(this);
    amplitude.input(mySound);
}

void draw() {
    background(0);
    stroke(255);
    noFill();
    
    waveform = amplitude.getWaveform();
    beginShape();
    for (int i = 0; i < waveform.length; i++) {
        float x = map(i, 0, waveform.length, 0, width);
        float y = map(waveform[i], -1, 1, height / 2, 0);
        vertex(x, y);
    }
    endShape();
}

Explanation:

  • amplitude.getWaveform();: Retrieves the waveform data of the audio signal.
  • beginShape(); ... endShape();: Draws the waveform data on the screen.

This tutorial demonstrates how to create a real-time waveform visualizer for audio analysis.

Tutorial 3: Implementing a Real-Time Beat Detection

Objective: Detect and visualize beats in real-time from an audio stream.

1. Detecting Beats:


import processing.sound.*;

SoundFile mySound;
BeatDetect beatDetect;
int beatThreshold = 10;

void setup() {
    size(800, 600);
    mySound = new SoundFile(this, "soundfile.wav");
    mySound.play();
    beatDetect = new BeatDetect(this);
    beatDetect.input(mySound);
}

void draw() {
    background(255);
    beatDetect.process();
    
    if (beatDetect.isBeat(beatThreshold)) {
        fill(255, 0, 0);
        ellipse(width / 2, height / 2, 100, 100); // Indicate beat with a red circle
    }
}

Explanation:

  • beatDetect.process();: Processes the audio to detect beats.
  • beatDetect.isBeat(beatThreshold);: Checks if the detected beat exceeds the threshold.

This tutorial covers how to implement beat detection and visualize it in real-time.

Tutorial 4: Designing a Real-Time Audio Reverb Effect

Objective: Create a real-time audio reverb effect to enhance sound quality.

SoundFile mySound;
Reverb reverb;

Explanation:

  • reverb.process(mySound, reverbAmount);: Applies reverb effect with the updated amount.

This tutorial shows how to design a real-time audio reverb effect and control it interactively.

Tutorial 5: Creating a Dynamic Sound Effects Processor

Objective: Design a sound effects processor with dynamic parameter control.

1. Implementing Sound Effects Processing:


import processing.sound.*;

SoundFile mySound;
Filter filter;
Delay delay;
float filterCutoff = 1000;
float delayTime = 0.5;

void setup() {
    size(800, 600);
    mySound = new SoundFile(this, "soundfile.wav");
    mySound.play();
    filter = new Filter(this);
    delay = new Delay(this);
}

void draw() {
    background(255);
    filterCutoff = map(mouseX, 0, width, 100, 5000); // Control filter cutoff with mouseX
    delayTime = map(mouseY, 0, height, 0.1, 1); // Control delay time with mouseY
    
    filter.process(mySound, filterCutoff, 1);
    delay.process(mySound, delayTime);
}

Explanation:

  • filterCutoff = map(mouseX, 0, width, 100, 5000);: Adjusts filter cutoff frequency based on mouseX position.
  • delayTime = map(mouseY, 0, height, 0.1, 1);: Adjusts delay time based on mouseY position.
  • filter.process(mySound, filterCutoff, 1);: Applies the filter with the updated cutoff frequency.
  • delay.process(mySound, delayTime);: Applies the delay with the updated time parameter.

This tutorial demonstrates how to create a dynamic sound effects processor with real-time control.

Tutorial 7: Creating an Interactive Drum Machine

Objective: Design a simple drum machine with interactive controls.

1. Implementing a Drum Machine:


import processing.sound.*;

SoundFile kick, snare, hiHat;
boolean kickPlaying = false;
boolean snarePlaying = false;
boolean hiHatPlaying = false;

void setup() {
    size(800, 600);
    kick = new SoundFile(this, "kick.wav");
    snare = new SoundFile(this, "snare.wav");
    hiHat = new SoundFile(this, "hihat.wav");
}

void draw() {
    background(255);
    if (mousePressed) {
        if (mouseX < width / 3) {
            kick.play();
        } else if (mouseX < 2 * width / 3) {
            snare.play();
        } else {
            hiHat.play();
        }
    }
}

Explanation:

  • kick.play();, snare.play();, hiHat.play();: Plays different drum sounds based on mouse position.

This tutorial introduces creating an interactive drum machine, where different drum sounds are triggered by mouse clicks.

Tutorial 8: Designing a Real-Time Audio Spectrum Analyzer

Objective: Visualize the frequency spectrum of an audio signal in real-time.

1. Creating a Spectrum Analyzer:


import processing.sound.*;

SoundFile mySound;
FFT fft;
int spectrumSize = 512;

void setup() {
    size(800, 600);
    mySound = new SoundFile(this, "soundfile.wav");
    mySound.play();
    fft = new FFT(this, spectrumSize);
    fft.input(mySound);
}

void draw() {
    background(0);
    fft.analyze();
    stroke(255);
    noFill();
    beginShape();
    for (int i = 0; i < fft.specSize(); i++) {
        float x = map(i, 0, fft.specSize(), 0, width);
        float y = map(fft.getBand(i), 0, 1, height, 0);
        vertex(x, y);
    }
    endShape();
}

Explanation:

  • fft.analyze();: Analyzes the audio input to get frequency spectrum data.
  • beginShape(); ... endShape();: Visualizes the frequency spectrum as a waveform.

This tutorial covers creating a real-time audio spectrum analyzer for visualizing frequency data.

Tutorial 10: Designing a Real-Time Audio Filter Bank

Objective: Create a filter bank to apply multiple filters to an audio signal.

1. Implementing a Filter Bank:


import processing.sound.*;

SoundFile mySound;
Filter[] filters;
int numFilters = 5;

void setup() {
    size(800, 600);
    mySound = new SoundFile(this, "soundfile.wav");
    mySound.play();
    filters = new Filter[numFilters];
    for (int i = 0; i < numFilters; i++) {
        filters[i] = new Filter(this);
    }
}

void draw() {
    background(255);
    for (int i = 0; i < numFilters; i++) {
        float freq = map(i, 0, numFilters, 100, 10000);
        filters[i].process(mySound, freq, 1);
    }
}

Explanation:

  • filters[i].process(mySound, freq, 1);: Applies each filter with a different frequency.

This tutorial demonstrates how to design a filter bank and apply multiple filters to an audio signal.
