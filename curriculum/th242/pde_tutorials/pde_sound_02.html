Sound Design Tutorial Series 2

Tutorial 1: Building a Simple Synthesizer

Objective: Create a basic synthesizer with multiple sound parameters.

1. Implementing a Multi-Parameter Synthesizer:


import processing.sound.*;

Oscillator osc;
Envelope env;
float freq = 440;
float amp = 0.5;

void setup() {
    size(800, 600);
    osc = new Oscillator(this);
    env = new Envelope(this);
    osc.play();
}

void draw() {
    background(255);
    freq = map(mouseX, 0, width, 100, 1000); // Frequency control
    amp = map(mouseY, 0, height, 0, 1); // Amplitude control
    osc.setFreq(freq);
    env.set(amp, 0.1, 0.2, 0.3); // ADSR envelope
    osc.amp(env);
}

Explanation:

  • osc.setFreq(freq);: Sets the oscillator frequency based on mouseX position.
  • env.set(amp, 0.1, 0.2, 0.3);: Configures the ADSR envelope with attack, decay, sustain, and release times.
  • osc.amp(env);: Applies the envelope to the oscillator's amplitude.

This tutorial demonstrates how to build a simple synthesizer with interactive frequency and amplitude controls.

Tutorial 2: Creating an Interactive Sound Visualizer

Objective: Visualize sound parameters in real-time with graphical representations.

1. Visualizing Sound Waves:


import processing.sound.*;

SoundFile mySound;
Oscillator osc;

void setup() {
    size(800, 600);
    osc = new Oscillator(this);
    osc.play();
    mySound = new SoundFile(this, "soundfile.wav");
    mySound.play();
}

void draw() {
    background(0);
    stroke(255);
    noFill();
    
    float[] waveform = osc.getWaveform();
    beginShape();
    for (int i = 0; i < waveform.length; i++) {
        float x = map(i, 0, waveform.length, 0, width);
        float y = map(waveform[i], -1, 1, 0, height);
        vertex(x, y);
    }
    endShape();
}

Explanation:

  • float[] waveform = osc.getWaveform();: Retrieves the current waveform data from the oscillator.
  • beginShape(); ... endShape();: Draws the waveform data as a continuous line on the screen.

This tutorial shows how to visualize sound waves with real-time graphical representations.

Tutorial 3: Building a Real-Time Audio Effects Processor

Objective: Create a real-time audio effects processor with multiple effects.

1. Implementing Multiple Audio Effects:


import processing.sound.*;

SoundFile mySound;
Reverb reverb;
Delay delay;
Filter filter;

void setup() {
    size(800, 600);
    mySound = new SoundFile(this, "soundfile.wav");
    mySound.play();
    reverb = new Reverb(this);
    delay = new Delay(this);
    filter = new Filter(this);
    
    reverb.process(mySound, 0.5); // Apply reverb
    delay.process(mySound, 0.5, 0.5); // Apply delay
    filter.process(mySound, 1000, 1); // Apply filter
}

void draw() {
    background(255);
}

Explanation:

  • Reverb, Delay, Filter: Creates effect objects for reverb, delay, and filter.
  • reverb.process(mySound, 0.5);: Applies reverb to the sound.
  • delay.process(mySound, 0.5, 0.5);: Applies delay with feedback and delay time.
  • filter.process(mySound, 1000, 1);: Applies a filter with a cutoff frequency and resonance.

This tutorial introduces combining multiple audio effects for a richer sound processing experience.

Tutorial 4: Developing a Real-Time Audio Recorder

Objective: Create an audio recorder that captures and plays back sound in real-time.

1. Implementing Audio Recording and Playback:


import processing.sound.*;

AudioIn input;
SoundFile recorder;

void setup() {
    size(800, 600);
    input = new AudioIn(this, 0);
    input.start();
    recorder = new SoundFile(this, "recorded.wav");
}

void draw() {
    background(255);
    if (mousePressed) {
        recorder.record(input); // Start recording
    } else {
        recorder.stop(); // Stop recording
    }
}

Explanation:

  • AudioIn input;: Captures audio input from the microphone.
  • recorder.record(input);: Starts recording the audio input.
  • recorder.stop();: Stops recording and saves the audio file.

This tutorial shows how to capture and record audio in real-time, with basic controls for starting and stopping recording.

Tutorial 5: Creating a Simple Audio Sequencer

Objective: Develop a basic audio sequencer to trigger sounds in a sequence.

1. Implementing a Basic Sequencer:


import processing.sound.*;

SoundFile[] sounds;
int[] sequence = {0, 1, 2, 0}; // Sequence of sounds
int step = 0;
int tempo = 120; // Beats per minute
int interval;

void setup() {
    size(800, 600);
    sounds = new SoundFile[3];
    sounds[0] = new SoundFile(this, "sound1.wav");
    sounds[1] = new SoundFile(this, "sound2.wav");
    sounds[2] = new SoundFile(this, "sound3.wav");
    interval = 60000 / tempo; // Interval in milliseconds
    frameRate(1 / (interval / 1000.0)); // Set frame rate to match tempo
}

void draw() {
    background(255);
    if (millis() % interval < 50) {
        sounds[sequence[step]].play(); // Play sound according to sequence
        step = (step + 1) % sequence.length; // Move to next step
    }
}

Explanation:

  • int[] sequence = {0, 1, 2, 0};: Defines the sequence of sounds to be played.
  • interval = 60000 / tempo;: Calculates the interval between steps based on the tempo.
  • sounds[sequence[step]].play();: Plays the sound at the current step of the sequence.

This tutorial helps you build a simple audio sequencer that plays a sequence of sounds at a specified tempo.

Tutorial 7: Building a Simple Audio Filter Bank

Objective: Create a filter bank with multiple filters to process audio in real-time.

1. Implementing Multiple Filters:


import processing.sound.*;

SoundFile mySound;
Filter lowPass;
Filter highPass;
Filter bandPass;

void setup() {
    size(800, 600);
    mySound = new SoundFile(this, "soundfile.wav");
    mySound.play();
    lowPass = new Filter(this);
    highPass = new Filter(this);
    bandPass = new Filter(this);
    
    lowPass.process(mySound, 500, 0.5); // Low-pass filter
    highPass.process(mySound, 2000, 0.5); // High-pass filter
    bandPass.process(mySound, 1000, 1); // Band-pass filter
}

void draw() {
    background(255);
}

Explanation:

  • Filter lowPass, highPass, bandPass;: Creates different filter objects.
  • lowPass.process(mySound, 500, 0.5);: Applies a low-pass filter with a cutoff frequency of 500 Hz.
  • highPass.process(mySound, 2000, 0.5);: Applies a high-pass filter with a cutoff frequency of 2000 Hz.
  • bandPass.process(mySound, 1000, 1);: Applies a band-pass filter with a cutoff frequency of 1000 Hz.

This tutorial shows how to implement a filter bank with multiple filters to process audio in various ways.

Tutorial 8: Creating a Dynamic Soundscape Generator

Objective: Develop a soundscape generator that creates evolving audio environments.

1. Generating Evolving Soundscapes:


import processing.sound.*;

Oscillator osc1;
Oscillator osc2;

void setup() {
    size(800, 600);
    osc1 = new Oscillator(this);
    osc2 = new Oscillator(this);
    osc1.play();
    osc2.play();
}

void draw() {
    background(255);
    float freq1 = map(sin(millis() * 0.001), -1, 1, 200, 800);
    float freq2 = map(cos(millis() * 0.001), -1, 1, 300, 900);
    
    osc1.setFreq(freq1);
    osc2.setFreq(freq2);
    
    osc1.amp(0.5);
    osc2.amp(0.5);
}

Explanation:

  • float freq1 = map(sin(millis() * 0.001), -1, 1, 200, 800);: Uses a sine function to modulate the frequency of the first oscillator.
  • float freq2 = map(cos(millis() * 0.001), -1, 1, 300, 900);: Uses a cosine function to modulate the frequency of the second oscillator.

This tutorial demonstrates how to create a dynamic soundscape generator by evolving sound parameters over time.

Tutorial 9: Implementing a Real-Time Audio Spectrum Analyzer

    }
    endShape();
}

Explanation:

  • amplitude.getSpectrum();: Retrieves the spectrum data of the audio signal.
  • beginShape(); ... endShape();: Draws the spectrum data as a graphical representation on the screen.

This tutorial shows how to visualize the audio spectrum using a real-time analyzer.

Tutorial 10: Designing an Interactive Audio Effects Interface

Objective: Create an interactive interface to control audio effects in real-time.

1. Building an Audio Effects Control Panel:


import processing.sound.*;

SoundFile mySound;
Reverb reverb;
Filter filter;
float reverbAmount = 0.5;
float filterCutoff = 1000;

void setup() {
    size(800, 600);
    mySound = new SoundFile(this, "soundfile.wav");
    mySound.play();
    reverb = new Reverb(this);
    filter = new Filter(this);
    reverb.process(mySound, reverbAmount);
    filter.process(mySound, filterCutoff, 1);
}

void draw() {
    background(255);
    reverbAmount = map(mouseX, 0, width, 0, 1); // Control reverb amount with mouseX
    filterCutoff = map(mouseY, 0, height, 100, 5000); // Control filter cutoff with mouseY
    reverb.process(mySound, reverbAmount);
    filter.process(mySound, filterCutoff, 1);
}

Explanation:

  • reverbAmount = map(mouseX, 0, width, 0, 1);: Controls the reverb amount based on mouseX position.
  • filterCutoff = map(mouseY, 0, height, 100, 5000);: Controls the filter cutoff frequency based on mouseY position.
  • reverb.process(mySound, reverbAmount);: Applies the updated reverb amount.
  • filter.process(mySound, filterCutoff, 1);: Applies the updated filter cutoff frequency.

This final tutorial shows how to create an interactive audio effects interface for real-time sound manipulation.
