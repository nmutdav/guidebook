Sound Design Tutorial Series 1

Tutorial 1: Introduction to Sound in Processing

Objective: Learn the basics of integrating sound with Processing.

1. Setup Processing for Sound

  • Install the Sound Library:
    • Open Processing.
    • Go to Sketch > Import Library > Add Library....
    • Search for "Sound" and install it.
  • Play a Sound File:

import processing.sound.*;

SoundFile mySound;

void setup() {
    size(800, 600);
    mySound = new SoundFile(this, "soundfile.mp3");
    mySound.play();
}

void draw() {
    background(255);
}

Explanation:

  • import processing.sound.*;: Imports the Sound library for sound functionalities.
  • SoundFile mySound;: Declares a variable for the sound file.
  • void setup(): Initializes the sketch and plays the sound file.
  • mySound = new SoundFile(this, "soundfile.mp3");: Loads the sound file. Replace "soundfile.mp3" with your sound file.
  • mySound.play();: Starts playback of the sound file.

This tutorial introduces how to play a sound file in Processing. Ensure that the sound file is in the sketch’s "data" folder.

Tutorial 2: Controlling Sound Playback

Objective: Learn to control sound playback with basic interactions.

1. Play and Pause Sound:


import processing.sound.*;

SoundFile mySound;
boolean isPlaying = false;

void setup() {
    size(800, 600);
    mySound = new SoundFile(this, "soundfile.mp3");
}

void draw() {
    background(255);
}

void mousePressed() {
    if (isPlaying) {
        mySound.pause();
    } else {
        mySound.play();
    }
    isPlaying = !isPlaying;
}

Explanation:

  • boolean isPlaying = false;: Tracks whether the sound is currently playing.
  • void mousePressed(): A function that is called when the mouse is pressed. It toggles play and pause of the sound.
  • if (isPlaying): Checks if the sound is currently playing to pause it or play it based on the current state.
  • mySound.pause(); and mySound.play();: Control the playback of the sound.

This tutorial demonstrates how to control sound playback using mouse interactions.

Tutorial 2: Creating Sound Envelopes

Objective: Learn to create sound envelopes to shape the attack, decay, sustain, and release of sounds.

1. Implementing an ADSR Envelope:


import processing.sound.*;

Oscillator osc;
float attack = 0.1;
float decay = 0.2;
float sustain = 0.7;
float release = 0.3;

void setup() {
    size(800, 600);
    osc = new Oscillator(this);
    osc.play();
}

void draw() {
    background(255);
    // Example of envelope parameters affecting sound
    osc.amp(1.0, attack); // Attack phase
    osc.amp(sustain, decay); // Decay and sustain phase
    osc.amp(0, release); // Release phase
}

Explanation:

  • osc.amp(1.0, attack);: Sets the amplitude to maximum during the attack phase.
  • osc.amp(sustain, decay);: Adjusts amplitude to the sustain level after decay.
  • osc.amp(0, release);: Fades out the sound during the release phase.

This tutorial introduces ADSR (Attack, Decay, Sustain, Release) envelopes for shaping sound characteristics.

Tutorial 5: Implementing Sound Modulation

Objective: Apply modulation techniques to change sound characteristics over time.

1. Using Frequency Modulation (FM):


import processing.sound.*;

Oscillator carrier;
Oscillator modulator;

void setup() {
    size(800, 600);
    carrier = new Oscillator(this);
    modulator = new Oscillator(this);
    carrier.play();
    modulator.play();
}

void draw() {
    background(255);
    float modFreq = map(mouseX, 0, width, 1, 10); // Modulator frequency
    modulator.setFreq(modFreq);
    carrier.setFreq(440 + modulator.getFreq() * 50); // Frequency modulation
}

Explanation:

  • modFreq = map(mouseX, 0, width, 1, 10);: Maps the mouseX position to a frequency range for modulation.
  • carrier.setFreq(440 + modulator.getFreq() * 50);: Applies frequency modulation to the carrier signal.

This tutorial demonstrates frequency modulation to create complex sounds by varying the carrier frequency.

Tutorial 9: Modulating Sound Parameters

Objective: Apply modulation techniques to dynamically change sound parameters.

1. Modulating Amplitude with a LFO:


import processing.sound.*;

Oscillator osc;
Oscillator lfo;

void setup() {
    size(800, 600);
    osc = new Oscillator(this);
    lfo = new Oscillator(this);
    lfo.setFreq(0.5); // Low-Frequency Oscillator
    lfo.amp(0.5);
    lfo.play();
    osc.play();
}

void draw() {
    background(255);
    osc.amp(0.5 + lfo.get() * 0.5); // Modulate amplitude with LFO
}

Explanation:

  • lfo.setFreq(0.5);: Sets the frequency of the low-frequency oscillator (LFO).
  • osc.amp(0.5 + lfo.get() * 0.5);: Modulates the amplitude of the main oscillator with the LFO.

This tutorial demonstrates how to use an LFO to modulate sound parameters such as amplitude.

Tutorial 10: Building a Simple Drum Machine

Objective: Create a basic drum machine using sound samples and triggers.

1. Triggering Drum Samples:


import processing.sound.*;

SoundFile kick;
SoundFile snare;
SoundFile hiHat;

void setup() {
    size(800, 600);
    kick = new SoundFile(this, "kick.wav");
    snare = new SoundFile(this, "snare.wav");
    hiHat = new SoundFile(this, "hihat.wav");
}

void draw() {
    background(255);
    if (mousePressed) {
        if (mouseX < width / 3) kick.play(); // Play kick drum
        else if (mouseX < width * 2 / 3) snare.play(); // Play snare drum
        else hiHat.play(); // Play hi-hat
    }
}

Explanation:

  • SoundFile kick, snare, hiHat;: Creates SoundFile objects for different drum sounds.
  • if (mousePressed) { ... }: Plays different drum sounds based on mouse position.

This tutorial helps you build a simple drum machine using sample playback and user input.
